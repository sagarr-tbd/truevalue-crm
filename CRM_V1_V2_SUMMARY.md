# TrueValue CRM — V1 & V2 Overview

## What is V1 and V2?

**V1 (Version 1)** is the original CRM system that is currently in use. It handles all sales, activities, and settings.

**V2 (Version 2)** is the next-generation upgrade built alongside V1. It introduces smarter data handling, customizable forms, and better tracking — without breaking anything in V1. Both versions run side by side so the transition is safe and gradual.

## Key Improvements in V2

### 1. Customizable Forms
In V1, every lead, contact, or deal has the same fixed set of fields. In V2, admins can **add, remove, and rearrange form fields** without needing a developer. Each entity type (Leads, Contacts, Companies, Deals) has its own form layout editor.

### 2. Smarter Data Storage
V2 uses a "hybrid" approach — essential fields (like name, email, status) are stored normally for fast searching, while custom fields are stored flexibly. This means you get **both speed and flexibility**.

### 3. Deal Stage History
V2 automatically tracks **every time a deal moves from one stage to another** — who moved it, when, and from which stage. This gives full visibility into the sales pipeline progression.

### 4. Better Pipeline Management
V2 pipelines support:
- **Win/Loss flags** on stages (mark which stages mean a deal is won or lost)
- **Rotting days** (get alerted when a deal sits too long in one stage)
- **Stage colors and probability percentages** for visual clarity

### 5. Web Form Capture
V2 supports **public-facing web forms** that can be embedded on websites. When someone fills out a form, it automatically creates a lead in the CRM.

### 6. Bulk Operations
V2 supports **bulk actions** — select multiple records and update or delete them all at once, saving time on repetitive work.

### 7. "Accounts" Renamed to "Companies"
The V1 term "Accounts" was renamed to "Companies" in V2 for clearer, more intuitive naming.

## What Stays the Same

- **V1 continues to work** — nothing is removed or broken
- **Same login and permissions** — your existing user roles apply to both V1 and V2
- **Same look and feel** — V2 pages follow the same design language as V1
- **Same sidebar navigation** — V2 sections appear alongside V1 sections
